# Laundry Mart

A simple HTML webpage for Laundry Mart that showcases laundry services, pricing details, and a booking form for customers.

## Features

- Welcome section with introduction
- Laundry service image
- List of services offered
- Price table for services
- Customer booking form
- Contact information in footer

## Technologies Used

- HTML5

## How to Run

1. Download or clone the project.
2. Open the `index.html` file in any web browser.
3. The Laundry Mart webpage will be displayed.

- Folding
- Stain Removal
